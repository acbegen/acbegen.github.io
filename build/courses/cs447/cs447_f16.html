Course Overview

Description: This course provides an introduction to the fundamental concepts in the design and implementation of computer networks, protocols and applications. Topics to be covered include network architectures, applications, network programming interfaces, transport layer, routing layer and link layer. Time permitting, we will overview wireless networking and protocols, too.
Prerequisites: CS 201
